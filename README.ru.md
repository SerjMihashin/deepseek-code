# DeepSeek Code

**AI-агент для разработки в терминале** на базе DeepSeek API.

Аналог Qwen Code, Claude Code, OpenAI Codex CLI — с открытым исходным кодом.

> 📖 Полный отчёт о разработке: [CHANGELOG.md](CHANGELOG.md)

## Возможности

### 🖥️ Интерактивный TUI
Полноценный терминальный интерфейс на базе Ink (React для терминала): история сообщений, автодополнение, горячие клавиши, отображение цепочки вызовов инструментов.

### 🤖 AI-агент с инструментами (NEW)
DeepSeek Code теперь работает как **настоящий AI-агент**! Вместо простого чата AI может:
- 📂 Читать файлы (`read_file`)
- ✏️ Создавать и редактировать файлы (`write_file`, `edit`)
- 🖥️ Выполнять shell-команды (`run_shell_command`)
- 🔍 Искать файлы по маске (`glob`) и содержимому (`grep_search`)
- 🌐 Работать с реальным браузером через нативный инструмент `chrome`

AI сам решает, какие инструменты использовать, итеративно выполняя задачи: прочитать → найти → исправить → проверить.

### 🔒 Режимы разрешений
| Режим | Описание |
|---|---|
| **Plan** | Только показывает, что сделает — ничего не выполняет |
| **Default** | Запрашивает подтверждение на каждое действие |
| **Auto-Edit** | Автоматически применяет изменения файлов, но не запускает команды |
| **YOLO** | Всё делает без подтверждения |

Переключение: `Tab`

### 🧠 Система памяти
- `/remember <текст>` — сохранить информацию в память
- `/forget <запрос>` — удалить из памяти
- `/memory` — показать все сохранённые воспоминания
- Автоматическое сохранение сессий

### 🔧 Инструменты (вызываются AI-агентом)
- **Read** — чтение файлов с offset/limit
- **Write** — создание и запись файлов
- **Edit** — замена текста (с replace_all)
- **Bash** — выполнение shell-команд
- **Glob** — поиск файлов по маске
- **Grep** — поиск по содержимому (ripgrep)
- **Chrome** — управление браузером Google Chrome: открытие страниц, клики, заполнение форм, выполнение JavaScript, чтение консоли, перехват сетевых запросов, скриншоты (NEW)

AI самостоятельно решает, какие инструменты вызывать, и итеративно выполняет задачи. В TUI отображается цепочка вызовов с иконками статуса: ⏳ → 🔄 → ✅ / ❌.

### 🔌 Продвинутые интеграции
- **MCP** (Model Context Protocol) — подключение внешних серверов инструментов
- **Subagents** — специализированные AI-агенты с изолированным контекстом
- **Skills** — модульные навыки через SKILL.md файлы
- **Hooks** — Pre/Post обработчики для инструментов
- **LSP** — Language Server Protocol (Go-to-Definition, Find References, Hover)

### 📋 Code Review
- `/review` — многошаговый пайплайн: линтеры + AI-анализ
- Оценка кода по шкале 0–100
- Поиск ошибок, уязвимостей, проблем производительности

### 📦 Git-интеграция
- `/git commit` — создание коммита
- `/git branch` — создание ветки
- `/git diff` — просмотр изменений
- `/git status` — статус репозитория

### 🏖️ Sandbox
- `/sandbox <команда>` — изолированное выполнение через Docker
- Автоматический fallback на прямой запуск если Docker недоступен

### ⏰ Планировщик
- `/loop 5m "проверить сборку"` — повторяющиеся задачи
- `/loop list` — список активных задач
- `/loop clear` — очистить все задачи

### 🎨 Темы оформления
- `/theme` — список тем
- `/theme <имя>` — переключение
- Встроенные темы: default, light, dracula, nord, solarized
- Пользовательские темы: `.deepseek-code/themes/*.json`

### 🌐 Локализация
- `/lang` — список языков
- `/lang ru` — переключение на русский
- Поддержка: English, Русский, 中文

### 🧩 Расширения
- Плагинная система через `.deepseek-code/extensions/<name>/package.json`
- Возможность добавлять собственные команды и инструменты

### 📄 Чекпоинты
- `/checkpoint` — сохранить состояние изменений
- `/restore` — восстановить из чекпоинта

### 🤖 Headless / CI/CD
- `--json` — JSON-вывод для интеграции с CI/CD (включает tool_calls статистику)
- `--headless` — режим без TUI (pipe-friendly), с полной поддержкой tool calling
- `dsc -p "задача" --json` — однострочный запрос с JSON-ответом

## Установка

### Вариант 1: из npm (готовый пакет)

```bash
npm install -g deepseek-code
```

> **Важно:** после установки может потребоваться переоткрыть терминал, чтобы команда `deepseek-code` стала доступна.

### Вариант 2: из исходников (для разработки)

```bash
# Клонировать репозиторий
git clone https://github.com/deepseek-ai/deepseek-code.git
cd deepseek-code

# Установить зависимости
npm install

# Собрать проект
npm run build

# Установить глобально (для команды deepseek-code / dsc)
npm install -g .

# На Windows с nvm может потребоваться создать cmd-файлы вручную:
#   C:\nvm4w\nodejs\deepseek-code.cmd
#   @echo off
#   node "%~dp0node_modules\deepseek-code\dist\cli\index.js" %*
```

### Настройка API-ключа

Перед запуском необходимо указать API-ключ DeepSeek. Способы (в порядке приоритета):

```bash
# 1. Переменная окружения (рекомендуется)
set DEEPSEEK_API_KEY=sk-ваш_ключ

# 2. Файл .env в корне проекта
echo DEEPSEEK_API_KEY=sk-ваш_ключ > .env

# 3. Файл settings.json
#   ~/.deepseek-code/settings.json  (глобально)
#   .deepseek-code/settings.json    (в проекте)
```

## Использование

```bash
# Интерактивная сессия
deepseek-code

# Короткий алиас
dsc

# Однострочный запрос
dsc -p "Объясни структуру проекта"

# Запрос + продолжение в интерактиве
dsc -i "Давай отрефакторим этот модуль"

# Выбрать модель
dsc -m deepseek-reasoner

# YOLO режим
dsc -y

# Продолжить последнюю сессию
dsc -c

# JSON-режим для CI/CD
dsc -p "Найди все ошибки" --json

# Смена темы при запуске
dsc --theme dracula

# Смена языка при запуске
dsc --lang ru
```

## Конфигурация

Приоритет загрузки (от высшего к низшему):
1. Переменные окружения: `DEEPSEEK_API_KEY`, `DEEPSEEK_BASE_URL`, `DEEPSEEK_MODEL`
2. Проектный конфиг: `.deepseek-code/settings.json`
3. Пользовательский конфиг: `~/.deepseek-code/settings.json`
4. Значения по умолчанию

### Пример settings.json

```json
{
  "apiKey": "sk-...",
  "baseUrl": "https://api.deepseek.com",
  "model": "deepseek-chat",
  "approvalMode": "default",
  "temperature": 0.7,
  "maxTokens": 4096
}
```

## Архитектура проекта

```
src/
  cli/          — Точка входа, Commander CLI, headless-режим
  core/         — Память, сессии, чекпоинты, MCP, subagents, skills,
                  hooks, LSP, темы, i18n, расширения, ревью, sandbox,
                  git-интеграция, планировщик
  config/       — Загрузка настроек, значения по умолчанию
  api/          — DeepSeek API клиент
  tools/        — Инструменты (Read, Write, Edit, Bash, Glob, Grep)
  ui/           — Ink/React компоненты
  utils/        — Утилиты (логгер, .deepseekignore)
```
